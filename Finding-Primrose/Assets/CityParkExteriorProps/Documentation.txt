Update v5.0 -


Very significant update focused on optimization. Custom textures that weren't adding much uniqueness were replaced with tiling textures (Universal Materials). It is highly recommended to import into a new empty Unity project, or to make a copy of your project before updating to the new version to avoid any issues. 

All models have been improved with more consistant geometry to make everything more smooth and less polygonal looking. 

Tiling texture sets (PBR) were added to the current collection to increase the amount of options in the "Universal Materials" folder. Currently sitting at 188 Universal Materials. 

Previous trees were using tree creator and were no longer working in newer versions of Unity, so they were replaced with custom trees using LODs. The grass was also improved to include grass patches that can be placed as a tree on the terrain. 

Demo scenes, day and night, have been improved and reworked with all the updates and better lighting. 

Post-processing is used in the demo scenes and the profiles for day/night are available in their corresponding folders. Day/night camera prefabs are created for easy implementation in scenes with those profiles already connected.
